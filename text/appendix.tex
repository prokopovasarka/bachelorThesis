\section{Smazání 3 barev}
\paragraph{Prerekvizity: }
Uživatel má uložených 3 a více barev v knihovně.
\paragraph{Testovací scénář}\mbox{}\\
\begin{enumerate}
  \item Zapněte aplikaci a počkejte na načtení hlavní obrazovky či projděte nápovědu.
  \item Klikněte na tlačítko Library.
  \item Dlouhým stiskem vyberte libovolnou barvu.
  \item Krátkým stiskem vyberte dvě další barvy.
  \item Klikněte na Delete.
\end{enumerate}
